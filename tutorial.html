Как пользоваться симулятором

У программы есть несколько режимов.

Добавление тел
  1. Нажмите кнопку "режим добавления"
  2. Кликните в том месте, где хотите создать тело
  3. Кликните в том месте, где будет оканчиваться вектор начальной скорости тела
  4. Повторно нажмите на кнопку режима, чтобы выйти из него и запустить симуляцию
Вы также можете запустить уже готовые симуляции, нажав на кнопку "готовые симуляции" и выбрав из списка.

Перемещение, отдаление, приближение
Для перемещения по игровому полю используйте стрелочки на клавиатуре. Для приближения или отдаления вращайте колёсико мыши.

Изменение
Для изменения игровых объектов нажмите "режим выбора" и кликните на нужное тело. Слева появятся параметры объекта, которые вы можете изменять. Повторно нажмите на кнопку режима, чтобы из него выйти.

Удаление
Чтобы удалить все объекты на игровом поле, нажмите "удалить все объекты".

Запись видео
Чтобы начать видеосъёмку игрового поля нажмите "начать запись видео". Нажмите кнопку повторно для завершения записи.
